## Michi Yamamoto

I build AI systems that model how ideas are interpreted, evaluated, and acted on.

### Focus

- Multi-agent systems for evaluation and simulation
- Decision-support tools for creators and founders

### Selected Projects

  Test how the same idea lands across different audiences

- **MeaningMap**  
  Map interpretation space and surface divergence

- **Backlash Predictor**  
  Identify potential negative reactions before publishing

- **Drift Guard**  
  Detect architectural drift through multi-perspective analysis

### Approach
I’m interested in modeling human judgment by focusing on:
- How different people interpret the same input  
- Where alignment breaks down  
- How decisions emerge from competing perspectives
